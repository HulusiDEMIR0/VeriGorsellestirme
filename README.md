TR 

Python’da bulunan seaborn kütüphanesindeki veri setlerinden biri olan tips veri setini bu projede kullandık.
Veri setinden başlıca bahsetmek gerekirse bir restoranın günlük müşteri bilgilerinin tutulduğu bir veri setidir.
Veri setinde hem sayısal hem de kategorik değişkenlerin bulunması, bu veri setini görselleştirme tekniklerini uygulamak açısından zengin bir hâle getirmiştir.
Bu yüzden de bu veri setini kullanarak hem sayısal değişkenlerin hem de kategorik değişkenlerin nasıl görselleştirileceği hakkında fikir sunmaktadır.

EN 

We used the tips dataset from the seaborn library in Python for this project.
To briefly describe the dataset, it is a dataset that contains daily customer information from a restaurant.
The presence of both numerical and categorical variables in the dataset makes it rich in terms of applying visualization techniques.
For this reason, by using this dataset, it provides insight into how both numerical and categorical variables can be visualized.
